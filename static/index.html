Качество брендового спроса

Индекс намерения (лиды) — оформить, заказать, условия, тарифы, кэшбэк…
Сервисная нагрузка — войти, личный кабинет, приложение, активировать…
Индекс токсичности — закрыть, заблокировали, мошенники, обман, отзывы…
Качество тренда (TQI) — удержание спроса через 2 мес после пика

Структура спроса ОТП по интентам

Продуктовый (лиды) / сервисный (текущие клиенты) / токсичный / прочее · последние 12 мес. В демо — модельная оценка, в рабочем режиме считается по topRequests.
