# Wordstat Дашборд — статистика ключевых слов по регионам РФ

Минималистичный веб-дашборд для анализа статистики поисковых запросов через
**Yandex Wordstat API** ([документация](https://aistudio.yandex.ru/docs/ru/search-api/concepts/wordstat.html)).

## Возможности

- Поиск статистики по ключевой фразе.
- **Строгая разбивка по самостоятельным субъектам РФ.** Федеральные округа,
  Россия целиком и зарубежные регионы полностью исключаются из данных.
- Аналитические графики (Chart.js):
  - горизонтальная гистограмма показов по регионам (топ-15 / 30 / все);
  - линейный график динамики частоты **с выбором региона** (один субъект или
    все регионы РФ) за **последние 12 полных месяцев** (текущий незавершённый
    месяц не учитывается);
  - кольцевая диаграмма долей топ-10 регионов.
- Таблицы регионов (с фильтром) и связанных запросов.
- **Выгрузка в Excel** (.xlsx), лист «Регионы РФ»: без заголовочных строк и без
  столбца с номерами; на каждый регион — отдельная колонка по каждому из
  последних 12 полных месяцев. Также листы «Динамика» и «Связанные запросы».
- Полностью русскоязычный интерфейс в **фирменном стиле ОТП Банка** (зелёная
  палитра).
- **Демо-режим**: без ключа API показываются синтетические данные —
  интерфейс и выгрузка работают сразу.

## Запуск

Windows — двойной клик по `run.bat`, либо вручную:

```powershell
python -m pip install -r requirements.txt
python app.py
```

Откройте <http://127.0.0.1:5000>.

## Рабочий режим (реальный API)

1. Скопируйте `.env.example` в `.env`.
2. Заполните `YANDEX_API_KEY` (или `YANDEX_IAM_TOKEN`) и `YANDEX_FOLDER_ID`.
3. Перезапустите сервер. Бейдж в правом верхнем углу сменится на «рабочий режим».

Требуется роль `search-api.webSearch.user` в Yandex Cloud.

## Как обеспечивается исключение федеральных округов

В дереве регионов Wordstat структура такая:

```
Россия (225)
  └── Федеральный округ      ← агрегат, ИСКЛЮЧАЕТСЯ
        └── Субъект РФ        ← ОСТАВЛЯЕМ (область / край / республика / город ФЗ / АО)
```

Модуль [`regions.py`](regions.py) берёт прямых потомков узла «Россия» как
федеральные округа (исключает их) и оставляет только их потомков — субъекты
РФ. Фильтр `filter_to_subjects` отбрасывает любые строки ответа API, чьи id
не входят в список субъектов: тем самым округа, Россия целиком и зарубежные
регионы гарантированно не попадают ни в графики, ни в Excel.

## Структура проекта

| Файл | Назначение |
|------|------------|
| `app.py` | Flask-сервер: API `/api/search`, `/api/export`, отдача статики |
| `yandex_client.py` | Клиент Wordstat API + генератор демо-данных |
| `regions.py` | Дерево субъектов РФ и логика исключения федеральных округов |
| `static/` | Фронтенд (HTML / CSS / JS, Chart.js) |
